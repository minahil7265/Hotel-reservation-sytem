# Hotel Reservation System Database

## Overview

A comprehensive Hotel Reservation System Database developed using Oracle SQL and PL/SQL. This project demonstrates database design, data integrity, query optimization, and business logic implementation through stored procedures, triggers, views, and indexes.

## Features

### Database Design

* Relational database schema
* Primary and Foreign Key constraints
* Data integrity enforcement
* Normalized database structure

### PL/SQL Procedures

* Guest registration
* Reservation creation
* Room assignment
* Service management
* Payment processing
* Reservation cost calculation

### Database Triggers

* Double-booking prevention
* Automatic room status updates
* Service validation
* Room release on checkout
* Payment validation
* Guest deletion protection

### Views

* Current Occupancy View
* Guest Service History View

### Indexes

* Reservation optimization indexes
* Room lookup indexes
* Payment query optimization
* Service history optimization

### Advanced SQL Queries

* Revenue analysis
* Occupancy reporting
* Guest analytics
* Service utilization analysis
* Staff performance reporting
* Analytic and correlated subqueries

## Technologies Used

* Oracle Database
* SQL
* PL/SQL
